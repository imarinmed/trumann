# ``TrumannSemanticMatcher``

Advanced job matching using semantic analysis and company intelligence.

## Overview

The SemanticMatcher module provides intelligent job-candidate matching beyond simple keyword matching. It uses AI to understand job requirements semantically and provides company insights for better decision making.
