# ``TrumannResumeParser``

AI-powered resume parsing, ATS optimization, and job-specific tailoring.

## Overview

The ResumeParser module provides comprehensive resume analysis and optimization capabilities. It can parse resumes from various formats, analyze ATS compatibility, and generate optimized versions tailored to specific job descriptions.
